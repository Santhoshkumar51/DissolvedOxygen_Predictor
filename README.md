🌊 Dissolved Oxygen Prediction System

📌 Overview

A hybrid ML-DL system to predict dissolved oxygen (DO) levels in aquaculture for better water quality management and early warning alerts.

🎯 Objective

Predict DO levels
Provide early alerts for low oxygen
Support efficient aquaculture management

🧠 Method

Feature Selection: LightGBM
Prediction Model: BiSRU + Attention
Time-series forecasting using sensor data

🛠️ Technologies

Python, Pandas, NumPy, Scikit-learn, LightGBM, TensorFlow/Keras, Streamlit/Flask

📊 Features

Hybrid ML + DL model
DO prediction & visualization
Early warning alert system

▶️ Run

pip install -r requirements.txt
streamlit run dashboard.py
